Язык программирования KLAD

KLAD Programming Language

Мультипарадигмальный высокоуровневый язык программирования специального назначения с динамической строгой типизацией и автоматическим управлением памятью, ориентированный на повышение производительности разработчика, читаемости кода и его качества, а также на обеспечение переносимости написанных на нём программ.

A multi-paradigm high-level programming language for special purposes with dynamic strong typing and automatic memory management, focused on increasing developer productivity, code readability and quality, as well as ensuring the portability of programs written in it.
